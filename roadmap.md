# 🚀 План развития Fast HEX Editor (Roadmap)

## 📦 Этап 1: Стабилизация и Оптимизация (Текущий статус)
* **Исправление критических багов Release-сборки** — исправлены ошибки маршалинга CsWinRT (`set_Tag`), настроен файл `.csproj` для стабильной работы [успешно выполнено].
* **Кастомизация базового интерфейса** — добавлена кастомная иконка приложения (`AppWindow.SetIcon`), настроены шрифты семейств Consolas/Segoe UI для корректного отображения HEX-сетки [успешно выполнено].
* **Архитектура работы с большими файлами** — реализовано потоковое чтение через `LargeFileDataProvider` и UI-виртуализация через `ItemsRepeater` для мгновенного открытия файлов 10 ГБ+.

## 🛠 Этап 2: Инструменты редактирования и Безопасность (Ближайший приоритет)
* **Система отмены изменений (Undo / Redo)**:
  * Внедрение паттерна «Команда» (Command Pattern) для отслеживания каждого измененного байта.
  * Добавление горячих клавиш `Ctrl + Z` (Отменить) и `Ctrl + Y` (Повторить).
* **Безопасная история изменений (In-Memory Buffer)**:
  * Перенос логики редактирования из прямой перезаписи файла во временный словарь `Dictionary<long, byte> modifiedBytes`.
  * Файл на диске остается нетронутым до тех пор, пока пользователь явно не нажмет кнопку «Сохранить» (`Ctrl + S`).
  * Визуальное выделение измененных (но еще не сохраненных) байтов контрастным цветом (например, оранжевым).
* **Полноценный интерактивный поиск**:
  * Создание удобного UI для панели поиска (Ctrl+F) с возможностью переключения режимов: "Искать текст (ASCII/UTF-8)" или "Искать HEX-последовательность".
  * Подсветка всех найденных совпадений на экране и функция «Найти далее» (F3).

## 🎨 Этап 3: Интерфейс и Кастомизация Стилей
* **Собственные настройки стилей (Custom Themes)**:
  * Создание панели настроек, где пользователь может вручную настроить цвета для Светлой (Light) и Темной (Dark) темы.
  * Возможность выбора индивидуальных цветов для: фонда редактора, текста байтов, адресов смещения, выделенного текста и измененных байтов.
  * Сохранение конфигурации стилей пользователя в локальный файл настроек `settings.json`.
* **Интеграция с дизайном Windows 11**:
  * Включение эффекта `ExtendsContentIntoTitleBar` для переноса вкладок `TabView` прямо в шапку окна (как в Google Chrome или Windows Terminal).
  * Применение современных материалов Mica или Acrylic для адаптивного размытия фона приложения.
