Quick Start with BinStudio

        
        

            
1. Editing Mode

            
Double-click any byte. Type the first half-byte (the cell shifts into a ? state), then the second. Modified cells are highlighted in red until saved.

        


        

            
2. Clipboard Operations

            
Select a range of bytes by dragging LMB. Press Ctrl + C to export as a clean HEX string. Press Ctrl + V to paste with automatic spacing parsing.

        


        

            
3. Theme Switching

            
Navigate to View -> Theme menu to instantly switch between dark, light, or system themes. Your preferences are saved automatically.
